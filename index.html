Evolus-Pencil-HTML-Export-Template

An Evolus Pencil(http://pencil.evolus.vn/) HTML Export Template developed to simplify the viewing of your finished wireframes.

It mimics a bit the functionality of the first version if Axure html formats and uses jQuery (albeit, an old version - 1.3.2).

Features:

  • View content 1 page at a time.
  • Side navigation for pages in the left.
  • Bottom bar for page notes always visible

Based on:

  • jQuery template by Maxime Delorme
  • Pretotype template by Rudy Lattae

This is my first work on Evolus Pencil (was asked to create wireframes for a client) and being a developer, I could not help but tinker a bit with the export functionality.

Was a nice experience and will most probably enhance this plugin/template when I've got more time.

To Install:

  1. Download Left_and_bottom_panel_with_page_notes_0.1.zip
  2. Open the 'Evolus Pencil' Software
  3. Click on 'Tools' > 'Manage Export Template...' from the main menu
  4. Make sure that 'Exporting for HTML Documents' is selected.
  5. Click on 'Install New Template...'
  6. A file browser will open, navigate to the ZIP file in the 'bin' directory and select the file
  7. Close the 'Manage Export Template...' Screen, You will get a message confirming the installation

To Use:

  1. Once installed (see above), you can use this Template by exporting your Wireframe document.
  2. Open the 'Evolus Pencil' Software
  3. Click on 'Document' > 'Export Document...' from the main menu
  4. Select 'Single web page' and press 'Continue'
  5. Choose your pages to export, then press 'Continue'
  6. In Template, choose 'Left and Bottom panel with page notes'
  7. Select the other options as needed and also Specify where the output should be saved.
  8. Click 'Done', you will see the progress for the export and if everything is ok, your document will be exported where you specified.
  9. Open up your document by clicking on the 'index.html' page generated. You should see the new format for the output.
